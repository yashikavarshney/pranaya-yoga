# Pranaya - Yoga & Meditation App

A beautiful, mindful yoga companion built with React and Tailwind CSS.

## Features
- **Daily Flows**: Morning and evening yoga practices.
- **Voice of Yoga**: Community contest and storytelling.
- **Journaling**: Track your mindful journey.
- **Profile**: Monitor your streaks and progress.

## How to View
This project is a standalone HTML file. Simply open `index.html` in any web browser to see the app in action.
